SenseKit

SenseKit is the name of my final year project at Leeds. It is a customisable wireless sensor network developed in hardware and software. The SenseKit project is built to work with over 40 endpoint devices, each with the ability to wirelessly transmit information on light levels, temperature and humidity to a cenrtal control point.


This has been a year long project from specification to production and has demonstrated a wide range of skills. Alongside technical skills such as programming and design, project and time management skills have also been crucial.


The user interface has been built on the Flask python web framework with the front-end built using a combination of responsive HTML using the Bootstrap framework and javascript for the graphing of data. Hardware interfacing is written in python, utilising the PySerial library with variables stored in an SQLite database. The hardware is based on a Jennic/NXP JN5148 chip and runs on the IEEE 802.15.4 specification.
